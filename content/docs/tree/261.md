# 261. Graph Valid Tree 
[link](https://leetcode.com/problems/graph-valid-tree/)

**difficulty(1-5)**
3

**note**


## Description 
You have a graph of n nodes labeled from 0 to n - 1. You are given an integer n and a list of edges where edges[i] = [ai, bi] indicates that there is an undirected edge between nodes ai and bi in the graph.

Return true if the edges of the given graph make up a valid tree, and false otherwise.

 

Example 1:


Input: n = 5, edges = [[0,1],[0,2],[0,3],[1,4]]
Output: true
Example 2:


Input: n = 5, edges = [[0,1],[1,2],[2,3],[1,3],[1,4]]
Output: false
 

Constraints:

1 <= n <= 2000
0 <= edges.length <= 5000
edges[i].length == 2
0 <= ai, bi < n
ai != bi
There are no self-loops or repeated edges.

## Solution: BFS
to become a tree, it needs to meet 2 criteria:
1. node -1 = edge
2. all nodes are connected

```c++
class Solution {
public:
    bool validTree(int n, vector<vector<int>>& edges) {
        // criteria 1: n-1 == edges
        if (n == 0 || ((n -1) != edges.size()))
        {
            return false;
        }
        
        //make adjacent list
        vector<vector<int>> l(n);
        for (auto edge :edges)
        {
            l[edge[0]].push_back(edge[1]);
            l[edge[1]].push_back(edge[0]);
        }
        
        vector<bool> visited(n, false);
        queue<int> q;
        q.push(0);
        visited[0] = true;
        
        while (!q.empty())
        {
            int cur = q.front();
            q.pop();
            for (auto neighbor : l[cur])
            {
                if (visited[neighbor])
                {
                    continue;
                }
                q.push(neighbor);
                visited[neighbor] = true;
            }
        }
        
        // criteria 2: all nodes are visited
        for (auto node : visited)
        {
            if (node == false)
            {
                return false;
            }
        }
        return true;
    }
};
```
