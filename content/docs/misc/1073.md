# 1037. Valid Boomerang
[leetcode link](https://leetcode.com/problems/valid-boomerang/)

**difficulty(1-5)** 
1

**notes** 


## Description
A boomerang is a set of 3 points that are all distinct and not in a straight line.

Given a list of three points in the plane, return whether these points are a boomerang.

 

Example 1:

Input: [[1,1],[2,3],[3,2]]
Output: true
Example 2:

Input: [[1,1],[2,2],[3,3]]
Output: false
 

Note:

points.length == 3
points[i].length == 2
0 <= points[i][j] <= 100
 
## Solution
```c++
class Solution {
public:
    bool isBoomerang(vector<vector<int>>& points) {
        int xa = points[0][0];
        int ya = points[0][1];
        int xb = points[1][0];
        int yb = points[1][1];
        int xc = points[2][0];
        int yc = points[2][1];
        // 1. all distinct
        if ((xa == xb && ya == yb) 
           || (xa == xc && ya == yc)
           || (xb == xc && yb == yc)) {
            return false;
        }
        // 2. not straight line
        return (ya-yb)*(xa-xc) != (ya-yc)*(xa-xb);
        
        
    }
};
```
